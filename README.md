## µBlog

Este projeto compreende uma pequena aplicação Spring Boot de um blog simplista.

#### Tecnologias utilizadas no desenvolvimento foram:

* Java 8
* Spring Boot (1.5.7)
* JPA + H2 Database
* Thymeleaf
* Gradle
